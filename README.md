# Library App FE

Project frontend sederhana untuk katalog buku menggunakan Next.js dan Tailwind CSS.

---

## Fitur

- Menampilkan daftar buku  
- CRUD data buku (tambah, edit, hapus, detail)  

- Manajemen member  
- CRUD data member  

- Peminjaman buku  
- Mengembalikan buku  

- Menampilkan data peminjaman  
- Status buku (dipinjam / tersedia)  

---

## Instalasi dan Menjalankan Project

```bash
git clone https://github.com/dandyj18/library-fe.git
cd library-fe
npm install
npm run dev
